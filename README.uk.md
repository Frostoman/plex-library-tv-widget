# Бібліотека Plex — канали на головному екрані Android TV для вашої бібліотеки Plex

Легкий додаток для **Android TV**, що виводить вашу бібліотеку **[Plex](https://www.plex.tv/)** на
головний екран у вигляді **двох каналів** — один для **Фільмів**, один для **Серіалів**. Вибір
фільму **одразу запускає його** в застосунку Plex; вибір серіалу запускає його **перший
непереглянутий епізод**.

## Можливості

- 🎬 Два окремих **канали на головному екрані**: **Фільми** і **Серіали**.
- ▶️ Клік по **фільму** → він **запускається** в Plex. Клік по **серіалу** → стартує його **перший
  непереглянутий епізод** (переглянуті пропускаються).
- 🖼️ **Постери** для кожного тайтла.
- 🔗 **Вхід без клавіатури** через офіційний PIN-флоу **plex.tv/link**.
- 🖥️ **Вибір сервера** — якщо акаунт бачить кілька серверів Plex, можна обрати потрібний.
- 🔀 **Сортування**: *останні переглянуті* або *за алфавітом*.
- ✅ **Чекбокси для тайтлів** — вибір, що показувати в кожному каналі (за замовчуванням усе).
- ↻ **Плитка оновлення** в кінці кожного каналу + авто-оновлення при відкритті й у фоні.
- ✏️ Налаштовувані **назви каналів**.
- 🇬🇧🇺🇦 **Англійська** та **українська** локалізації.

## Як це працює

```
Цей додаток ──(PIN plex.tv/link)──► токен вашого акаунта Plex
   │  /api/v2/resources → знаходить ваш сервер Plex (+ токен сервера)
   │  /library/sections + /library/sections/{id}/all → фільми та серіали
   │  /photo/:/transcode → постери
   ▼
Два канали (Фільми + Серіали) на головному екрані Android TV
   │
   ▼ (клік по картці)
Фільм → грає в плеєрі Plex; Серіал → грає перший непереглянутий епізод
```

### Як клік по картці відкриває контент (перевірено на реальному Android TV)

[`PlexLauncher`](app/src/main/java/com/androidtv/plexwidget/launch/PlexLauncher.kt) використовує ті
самі інтенти, що й сам Plex:

- **Фільм** → `plex://server://{machineId}/com.plexapp.plugins.library/library/metadata/{ratingKey}`
  (той самий інтент, що несуть власні картки каналів Plex) → **одразу запускає відтворення**.
- **Серіал** → перший **непереглянутий** епізод знаходиться через `/library/metadata/{id}/allLeaves`
  (`viewCount == 0`) і запускається так само. Якщо все переглянуто — відкривається сторінка серіалу
  на `watch.plex.tv`.

Картки на головному екрані ведуть на невидимий трамплін
[`OpenItemActivity`](app/src/main/java/com/androidtv/plexwidget/ui/OpenItemActivity.kt), а не несуть
ці cross-package інтенти напряму — лаунчер Google TV підміняє пакет в інтентах, які запускає сам,
тож трамплін перевидає їх із нашого процесу.

> ⚠️ Ці deep-link спираються на внутрішні механізми застосунку Plex (не офіційний публічний API).
> Зараз працюють (перевірено на Android TV), але оновлення Plex теоретично може їх змінити.

## Вимоги

- Android TV **API 21+** (канали на головному екрані потребують **API 26+ / Android 8**).
- Встановлений застосунок **[Plex](https://play.google.com/store/apps/details?id=com.plexapp.android)** на телевізорі.
- **Акаунт Plex** з хоча б одним сервером Plex Media Server (бібліотеки Movies та/або TV Shows).

## Встановлення

1. Завантажте останній **`PlexLibraryAndroidTV-x.y.apk`** зі сторінки [Releases](../../releases).
2. Встановіть на Android TV одним зі способів:
   - **ADB:** `adb connect <IP-TV>:5555`, потім `adb install PlexLibraryAndroidTV-x.y.apk`, або
   - **USB / файловий менеджер:** скопіюйте APK на TV і відкрийте (увімкніть «встановлення з невідомих джерел»).

## Налаштування

1. На TV відкрийте **Бібліотека Plex** → ряд **Налаштування** → **Підключити акаунт Plex**.
2. З'явиться код. На телефоні чи ПК відкрийте **`plex.tv/link`**, увійдіть і введіть код.
3. Якщо акаунт має кілька серверів — **оберіть потрібний**.
4. Додаток автоматично синхронізує Фільми та Серіали.
5. Додайте канали **Фільми** та **Серіали** на головний екран через «Налаштувати канали» у лаунчері.
6. Опціонально, у **Налаштуваннях**: **Вибрати фільми / серіали**, **Сортування**, **Сервер**
   (перемкнути сервер), **Назви віджетів**.

## Збірка з вихідного коду

Потрібно: **JDK 17**, **Android SDK** (compile SDK 34). Gradle wrapper уже включений.

```bash
# Windows
gradlew.bat assembleDebug
# macOS / Linux
./gradlew assembleDebug
```

APK з'явиться за шляхом `app/build/outputs/apk/debug/PlexLibraryAndroidTV-debug.apk`.
Найпростіше відкрити проєкт у **Android Studio** (там уже є JDK).

Для **підписаної release-збірки** створіть `keystore.properties` у корені репозиторію (він у gitignore):

```properties
storeFile=release.keystore
storePassword=ваш-пароль
keyAlias=ваш-alias
keyPassword=ваш-пароль
```

…і запустіть `gradlew.bat assembleRelease`. Без цього проєкт усе одно збереться (release вийде без підпису).

## Відомі обмеження

- Deep-link спираються на **внутрішні механізми Plex** — зараз працюють, але це не гарантований публічний API.
- Тайтли без id сервера / каталожного slug (напр. неспівставлені «Other Videos») відкривають просто застосунок Plex.
- Деякі лаунчери додають до назви каналу мітку додатка з двокрапкою (напр. `Бібліотека Plex:`); це поведінка лаунчера.
